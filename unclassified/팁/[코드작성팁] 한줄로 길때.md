<br>

**코드 한 줄이 길어질 때**

<br>

```python
# 다음 코드 처럼 코드 한 줄이 길어져 가독성이 떨어질 경우
foo = long_function_name(var_one, var_two, var_three, var_four)
```

<br>

```python
# 첫 파라미터에 맞추어 정렬한다.
foo = long_function_name(var_one, var_two,
                         var_three, var_four)

# 한 단계 높은 들여쓰기로 표현한다.
foo = long_function_name(
    var_one, var_two,
    var_three, var_four)
```
